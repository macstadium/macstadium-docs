When using Orka for DevOps use cases, two license tiers are offered: Basic and Advanced.

### Basic License:

  * Orchestration among nodes with priority and grouping features.
  * OCI-compatible images, ensuring broad compatibility and flexibility.
  * K8S Native Interfaces, including kubectl and compatible tools, for a seamless Kubernetes experience.
  * Integration with Jenkins, GitHub Actions, BuildKite, and TeamCity for CI/CD pipelines and Packer support for creating images.



### Advanced License:

  * Everything in Basic+
  * Image Caching to optimize performance.
  * Network and VM Isolation for enhanced security.
  * Support for Linux Workloads, broadening the scope of applications.
